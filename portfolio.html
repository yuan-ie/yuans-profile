❥ Project Description:

Graphical User Interface for part of my Senior Design Project. Made to display breast tissue biopsy slide images (original image, annotated image, and results from multiple ML algorithms). Images in the directory can be browsed in the scrollbar window. When image is selected, user can view the image in original or zoomed display and results from different ML algorthims. Results include a histogram comparing True/False Positives/Negatives of the annotations vs. the results.

❥ Languages/Medium:

Python, PYQT6, Bash, Github.
